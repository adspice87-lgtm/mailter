SMTP / DKIM
Konfiguracja serwera sehosta + podpis DKIM dla lepszej dostarczalności.
🔧 Jak skonfigurować na sehoście (krok po kroku)
Wykonaj te kroki raz — potem appka sama wysyła maile przez Twój serwer.
1
Zaloguj się do panelu sehosta (DirectAdmin lub cPanel) i przejdź do Poczta → Zarządzaj kontami email. Utwórz skrzynkę np. newsletter@twojadomena.pl
2
Dane SMTP sehosta:
Host: mail.twojadomena.pl
Port: 587 (STARTTLS) lub 465 (SSL)
Login: pełny adres email skrzynki
3
DKIM — w panelu sehosta znajdź DNS → DKIM i włącz. Pobierz klucz prywatny i wklej poniżej. Rekord DNS TXT zostanie dodany automatycznie lub zrobisz to ręcznie:
mail._domainkey.twojadomena.pl → "v=DKIM1; k=rsa; p=KLUCZ"
4
Backend API — wgraj folder mailflow-backend na serwer i uruchom:
npm install cp .env.example .env nano .env # uzupełnij dane npm install -g pm2 pm2 start ecosystem.config.js pm2 save
API będzie dostępne na http://IP_SERWERA:3001
5
Firewall — odblokuj port 3001 na serwerze:
ufw allow 3001/tcp
Opcjonalnie: ustaw nginx jako reverse proxy żeby API było na api.twojadomena.pl
